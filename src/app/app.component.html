Auto Code Formatting

This project demonstrates how we can enable auto code formatting with Prettier and JS-Beautify using custom formatting configurations via Husky. This ensures automatic formatting on Git pre-commit hooks.

  • PR reviewers can focus on logic instead of unnecessary whitespace changes.
  • Global formatting rules are enforced in the remote repository, ensuring consistency regardless of local plugin installations.
  • Our pre-defined global code formatting rules will always be followed in our remote central repository, so it doesn't matter whether developers have installed these formatting plugins (Prettier, JS-Beautify) on their machine or not!
  • Save your development time without worrying about code formatting.
